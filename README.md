# Event Locator - server

### Prerequisites

You should install `Node`, `git`

### Installing

- `git clone https://github.com/alamariful1727/node-geojson-mongodb.git`
- `cd node-geojson-mongodb`
- `npm install`

### For development

- `npm run dev`

### For production

- `npm run build`
- `npm start`
